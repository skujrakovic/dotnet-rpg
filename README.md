# RPG Web API

Web API for a RPG game developed using .NET.

Udemy course: .NET 5 & .NET Core 3.1 Web API & Entity Framework Jumpstart

Used pattern: _Model-View-Controller_

Database: _SQL Server_

Authentication: _JSON Web Tokens_
